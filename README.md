# Library Management System

A full-stack Library Management System built using Spring Boot, PostgreSQL, Flutter, Docker, and Kubernetes.

## Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL

### Frontend
- Flutter
- Dart

### DevOps
- Docker
- Kubernetes

## Features

- Add books
- View books
- Update books
- Delete books
- Borrow/return management (planned)
- RESTful APIs
- Layered architecture

## Project Structure

library-management-system/
│
├── backend/
├── frontend/
├── docker/
├── kubernetes/
└── docs/

## Backend Architecture

Controller → Service → Repository → Database
